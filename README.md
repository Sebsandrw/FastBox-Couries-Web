# FastBox-Couries-Webc
FastBox Courier es un prototipo web para gestionar envíos e importaciones por peso. Permite registrar paquetes, calcular tarifas, rastrear pedidos, revisar productos y llevar una bitácora de las acciones realizadas dentro del sistema.  Fue desarrollado con HTML, CSS y JavaScript, usando datos simulados y almacenamiento local del navegador.
